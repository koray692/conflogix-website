Visual automation for jobs that do not need a full PLC stack.

CLC is designed for people who need reliable control logic but do not want to start from ladder software, licensing, and heavy configuration tools.

IN

Flow based logic

Create flows with input triggers, delays, output actions, PWM control, analog thresholds, and AND/OR conditions.

DO

Live control

Connect over serial or Wi-Fi, upload/download configurations, and monitor inputs, outputs, PWM, ADC, run/stop state, and active flows.

SAFE

Safe state behavior

Define output safe values for stop, communication loss, critical faults, or scenario actions that intentionally put the device into safe mode.
